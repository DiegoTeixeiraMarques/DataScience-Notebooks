1. Introdução

Este trabalho visa aplicar um modelo auto autorregressivo (AR) para prever o volume de entradas de processos para um cliente específico de um escritório de advocacia em Fortaleza-CE. Os dados foram rotulados para manter o anonimato.

2. Metodologia

Inicialmente foi realizado um tratamento nos dados para mantê-los numa granularidade mensal, depois foram vistos através de gráficos para entender sobre sua tendência e padronização. Por fim, aplica-se o modelo para prever os resultados e é calculado o erro quadrático médio para cada um.
